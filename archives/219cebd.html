秦阳：一个PPT主义者的成长故事

随笔
笔记
发布日期:   2022-10-13

这不是一本书，而是秦老师做的一次分享，内容十分精彩，推荐给大家观看。下面是我做的观看笔记。

https://v.qq.com/x/page/c038161348x.html

九大主义

  1. 结构化思考（思考能力）
  2. 视觉化表达（视觉能力）
  3. 故事化演绎（说写能力）
  4. 生活化学习（学习能力）
  5. 价值化连接（连接能力）
  6. 迭代化精进（迭代能力）
  7. 通用化延伸（延伸能力）
  8. 百分化体验（行动能力）
  9. 平台化整合（整合能力）

1. 结构化思考

关键词：#碎片化学习 #体系 #挖坑

如何提升自己的结构化思维能力？

体系 → 长文 → 写作（由浅到深的一个过程）

▌体系

如何构建自己的体系？

给自己挖坑，将信息填到坑中。我们给自己挖了什么坑，看到的就是什么。

护宇：如果只给自己挖一个坑，就很容易陷入单一思维。比如学习《工作型PPT》，如果只给自己挖了学习PPT技巧的坑，那你就学不到课程开发、以及如何把一个知识点讲清楚这类的知识。

如何构建一门知识的体系？

动态的意识，不断更新不断淘汰不断深化。

体系学习法：****构建体系 → 获取信息 → 实践体验 → 整理/纳入。这是一个循环，从哪步开始并不重要。

护宇：体系有时候不是一步到位的，可以迭代精进，先有个1.0版本，之后再有2.0版本、3.0版本……

▌长文

不知道写什么怎么办？

生活记录、书评影评、时事评论、收获感悟、经验总结、为图配文、留言互动、伙伴合作（来自严歌苓《一个写稿佬的生涯》）……

上课时，我们的同学都是坐成一个圈，十二个同学，老师坐在中间，然后他就说，某某某，你出一个词儿。被点名的同学先出一个名词，然后 老师叫第二个人说你接一个动词，接了一个动词以后他就说，用任何一 个你想到的最最独特的一个动词来让这个名词动起来。这样一种训练就 是首先是他告诉你什么能使文章变得非常有活力的、非常有动作的、非 常往前走的，走得比较快的是动词而不是形容词。

老师老是教我们看着它，直到看见它，看见它的形状，Do you smell anything？你闻到了吗，你嗅到了吗？调动的是你所有的感官，来把这 个故事往下进行。

为什么我现在写小说的画面感很强，我觉得这是跟我们学校的训练是很 有很有关系的。写一个东西要有质感。这段文字你写出的一个场景，要 有质感，最好还有触感，就是说有六种感觉都有，六种感官都有。”

——严歌苓《一个写稿佬的生涯》

这些怎么感觉都是零碎的东西？

长文章不是一蹴而就的，就像拍电影，不是一下子拍完了所有场景，而是抓住一条主线，把场景串起来。

提示：长文章可以是之前积累内容的串联过程，不是非要在一晚上就写完。

▌写书

把写书当成一个过程，而不是结果。

护宇：

老秦对于很多知识都有自己的分类文件夹，比如IP，社群营销等，而其中已经有6个真的实现了出书！

他会把所看所学的内容整合到每个文件夹中，等到抓住某个时机，比如近段时间IP很火，能够以非常快的速度出书。

在以出书的思维锻炼自己时，结构化思考的能力，相对于写长文提升更快，因为要注意的内容更多了，不再是像长文中只用关注几个段落，写书面对的章与章、章与全书的联系。

总结：多学思路，少记套路。通过结构视角观世界，带着独立思考过人生。

2. 视觉化表达

1）美是第一感受

2）岂止于美，重要的是让听众理解并记住

  • PPT辅助理解与记忆（制作完成后，问自己，我可以记住吗？）

3）除了PPT，也有很多东西需要视觉化

对别人视觉化：公开用PPT演讲的场合，少之又少，更多的是生活中的口头沟通与表达，纯粹的文字语言里藏着生动的场景画面。

对自己「视觉化」：增强自我激励的功效（讲到自己在参加全国比赛时，将全国冠军的头P成自己的，顺便还把老师也P进去了，让自我激励更加具象，心理作用↑）

4）如何提升自己的视觉化表达能力？

关键词：眼界（拓宽眼界，重复内化）

打开眼界，从一个点扩展出来，将自己的思维扩展

护宇：老秦由一个包装设计体会到图片裁剪/局部的力量，而非单纯地学到把一张图片放在幻灯片两边可以带来更好的展示效果。其实就是透过现象看到更深层次的东西。

Q：发现自己在看完之后还是很容易忘记……

A：举了小马哥从锅炉工到创意总监的故事。

收集精品 → 分类整理 → 烂熟于心 → 持续刺激

我们大多数人只做到前2步。

我用了一个笨办法，和同事用半个月的时间把德国的一本世界级广告创 意杂志十年来的作品从网上全部下载下来，一共是20000个顶尖的创意 作品，我又用了近一个月的时间，把他们分门别类整理成了10个PPT。

而我则把这20000个创意反复看了三遍以上。当我看完了这些创意后， 我发现，其实市面上大部分广告，创意方法都是来自这些经典的创意， 无非变变形式而已。

同时，我也收集了世界上最经典的文案，全部抄写了一遍。大部分经典 的文案我都是背诵或者能够够复述。这时，你写文案的时候就有了各种 不同题材和思路帮你。而我个人也在从事广告的6年间每天保持着阅读 10个以上广告案例的习惯。”

——小马宋《从锅炉工到创意总监》

提炼版：

1）收集十年来的20000个顶尖的创意作品

2）分门别类整理成了10个PPT

3）反复看了三遍以上

4）收集世界上最经典的文案，全部抄写了一遍

5）大部分经典的文案都能背诵或者复述

6）从事广告的6年间每天保持着阅读10个以上广告案例

3. 故事化演绎

会讲道理的人，永远打不过会讲故事的人。

如何提高讲故事的能力？

推荐书籍：《小说课》。

  1. 目标：主人公的目标是什么？
  2. 阻碍：他的阻碍是什么？
  3. 努力：他如何努力？
  4. 结果：结果如何？（通常是不好的）
  5. 意外：有超越努力的意外，可以改变这一切吗？
  6. 转弯：意外发生，情节如何转变？
  7. 结局：最后的结局是什么？

结果发现，积累了很多讲故事的模板，还是讲不好……

Why？缺素材！

故事的种类：

1）自己的：经历多、爱折腾、记性好（注意细节）

2）别人的：听的多、看的多、聊的多（讲出彩的）

3）虚构的：想象力、好奇、脑洞（要有圆满的逻辑，能够回到你要想讲的道理或某个点上）

总结：深入浅出，似非而是。从别人的演说中思考，在自己的故事里成长。

4. 生活化学习

正向：一开始借书看，发现大多都是技巧教学，学了不知道干嘛，比如如何改模板等，讲的东西不实用，不知如何应用（从0 → 1，看不到结果）

逆向：尝试从1 → 0，从后往前推，结果直观，保持学习过程不枯燥（有目标，知道自己在往这个点靠近，学习是为了呈现某个结果），每一步都在接近结果，兴趣和动力更足。

在哪里学习并不重要，关键在于了解自己的结果在哪里，能否拆解出要做的操作和基本功。

护宇：我觉得自己可以利用自己整理的PPT灵感库啊，这里面都是结果，我能从1到0的去学习。另外，说到拆解出要做的操作和基本功，《工作型PPT》这门课体现得淋漓尽致啊……

如何提高「生活化学习」的敏感度？

挖坑挖得深 + 学习体系建立 → 提高敏感度

总结：取之生活，用于生活有一颗持续学习的心态，就时刻拥有学习的平台。

5. 价值化连接

有一项拿的出手的本事，进行价值互换。问世界要什么之前，先想想我有什么，因此能换到什么。

对人一块一，对物九毛九（从PPT定制得到的启示）。一件东西值一块钱砍到九毛九，东西品质不会变，如果一个人的服务值，1.1砍到0.99，虽然成交了，得到的服务却可能变了（启示：应该给予应有的价值肯定）

「我有什么！」→ 「我要有粉丝量！」

提到自己用PPT技能、书籍，通过在微博课堂开课，转发打卡，达到一定量换书的形式吸引粉丝，随后成功勾搭到秋叶大叔的例子。

「我有什么！」→ 「我要进华艺！」

讲述自己进入华艺之前和在华艺工作的经历。进入华艺之前，通过一条微博读书笔记PPT（老杜出的一本书），成功勾搭到老杜，求职成功！在华艺工作，想学营销的知识，没人教，主动要求帮公司们做PPT（沟通、旁听学习）。

进了华艺工作之后，发现每天的主要工作就是做PPT。但老秦不满足于这样的现状，他想学关于营销市场方面的知识。但是没有人会主动教你的。所以他再次借用了自己会PPT的这个技能，主动帮同事做PPT。那做ppt的话，需要了解一些关于这个领域的知识，所以老秦就以此为缘由，参加这些同事的会议啊什么的，一开始就是旁听，后来懂得多了也能帮助做一些事情。

护宇：这里面有些细节挺有意思。老秦在读书笔记做完之后，并不是简单地发出去，@然后就完了。他@的人都是有选择的，要么跟老杜有关，要么跟这本书有关（比如这本书是湛庐文化出的，他也@了），而且粉丝量选择在5万以下（大V你是撬不动的，别人根本看不到你的互动）。

「我有什么！」→ 「我要扩圈子！」

护宇：老秦之前上过TED，TED里面有个教练组，他就说自己能够提供思维和PPT制作方面的东西，就加入进去了。每个去TED演讲的人，都要经过教练组的培训，借此机会认识了各行各业的精英。

总结：工具用到极致就是神器，自身价值方为连接底气。

6. 迭代化精进

如何提高不断迭代进步的「动力」？

1）兴趣爱好

2）为一个目标找到多重意义（动力 ↑，回报↑）

3）不服，是人类进步的意义（不服精神）

4）承诺，倒逼自己成长（承诺倒逼）

5）某些经历带来的思维习惯（思维习惯）

6）建加油站（找些东西激励自己）

举例：讲述自己出书的过程

打印了一本自己的自传 → 整理者 → 20页 → 秋叶 秦阳 → 秦阳 秋叶 → ？

写书最大的经验：快写慢改（建立全局观）。

护宇：比如老秦会在一周内把这个书全部写一遍，之后带着问题去吸收和借鉴自己所需要的东西。其实我帮老秦做事情也是啊。一开始在小密圈因为写东西逻辑性还可以，就开始写推文，之后接到修改《微信营销与运营》第2版修改这样的角色（可惜聊天记录没有了……以后还是要记得及时截图保存啊），现在是“时间管理”图书的整理者这样的角色，也许未来我也可以出一本自己的书呢，我觉得老秦会给我这样的机会的！

总结：持续复盘，积累复利（不断积累，获得充裕回报）每天进步一点点，每次接近一丢丢。

7. 通用化延伸

从哪些角度进行延伸？

横、纵、深

横向：

举例1：PPT
PPT → PS、AI、CDR……（不同的软件存在相似功能）

PPT：店铺装修、海报……（都涉及到版式等内容，不一定要用PS来呈现）

结构化思维、视觉化表达

举例2：微博

举例3：公众号

如何扩大自己的知识域？

1）增加知识点的连接次数（积极思考）

2）扩大知识点的储藏个数（拒绝单一思维）

如何成为一个斜杠青年？

成为斜杠青年最快的方法：深挖一个领域，延伸出很多的应用场景，然后进行交错。

总结：从点到面，纵横交错。挖掘背后支撑的底层逻辑，找到多元通用的应用场景

护宇：关于通用化延伸，更详细的说明

1、具体内容

延伸具体包括三个方面：横向 深向 纵向

1）横向：同类型的、类似的选题

2）深向：同选题的继续深挖，创作出更多的深度和高度

3）纵向：与其他领域进行碰撞，找到另外的跨界角度

总结：看到一个点，延伸出线，进而成为很多面。

2、案例展示（By 伟崇）

原选题：《PPT七大搞笑操作，你挨了几枪？》

横向：《Word七大搞笑操作，你又挨了几枪？》

深向：《PPT动画四大搞笑操作，你挨了几枪？》

原选题：《目录页万能设计公式》

横向：《封面页万能设计公式》、《转场页万能设计公式》、《结束页万能设计公式》

深向：《商务风目录页如何制作》、《学术风目录页设计大全》、《政党风目录页设计》

纵向：《跟网站学目录页的制作》、《从地铁海报学习到的目录页技巧》

原选题：《PPT图片4大处理细节》

横向：《PPT图标3大细节处理》、《PPT文字4大处理细节》、地图，表格，图表…..

纵向：《跟发布会学习图片处理》

8. 百分化体验

1）好的开始带动后续一连串的“好事情”

2）目标越高，所需时间越长，根据实际需求，制定合理目标

护宇：比如你只是想在工作中更顺手的使用PPT，那么几十个小时的学习完全够了，但是如果你想卖模板，或者接定制，那么你需要付出更多的努力

1%的意义何在？

最小行动。尝试1%，觉得可行，继续；不行，放弃；这是一个试错的过程

过去发生的一切都是今天的铺垫。

总结：走着走着，花就开了。对眼前的事全力以赴，顺其自然呈现下一步

9. 平台化整合

可能是工具的整合，人脉的整合、平台的整合

护宇：就像我写论文，会用到Excel、Word、知网、谷歌、印象笔记等。
